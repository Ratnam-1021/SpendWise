# SpendWise 💰

SpendWise is a premium personal finance tracker built with SwiftUI. It helps users track their daily expenses, visualize spending trends through charts, and manage their budget with ease.

## Features
- **Dashboard**: Quick overview of monthly spending and recent transactions.
- **Analytics**: Beautiful charts (Bar and Donut) to visualize spending by category and day.
- **Expense Management**: Easy-to-use interface to add, edit, and delete expenses.
- **Categorization**: Group expenses into categories with custom icons and colors.
- **Dark Mode**: Fully supports iOS dark and light modes.

## Tech Stack
- **UI**: SwiftUI
- **Database**: Core Data
- **Visuals**: Swift Charts
- **Architecture**: MVVM
